# Task Manager

Task Manager е малко конзолно приложение, разработено на Python.

## Функционалности

- Добавяне на задача
- Показване на всички задачи
- Маркиране на задача като завършена
- Изтриване на задача
- Търсене на задача
- Записване на задачите във файл

## Използвани технологии

- Python
- JSON
- Git
- GitHub

## Стартиране на проекта

```bash
python main.py
